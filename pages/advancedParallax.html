The scroll amount that is scrolled from the top of the wondow is pulled via Jquery using:

let Scroll = $(this).scrollTop()

This produces a value which we can transform the position of selected elements. the transform: translateX() here is an obvious candidate; but other effects such as opacity can be used. like on the title of this page.
