Basic Parallax Effect

This basic parallax scrolling is a pure css "trick" in which the background properties has been set to attachment: fixed. this makes the background not scroll but stay in place while the rest of the page scrolls over it. Important steps is the framing of the background. so that it remains static.

The fixed navigation is an important part of the experience. It should stay fixed to make the parallax feeling more apparent.
